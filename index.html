Abstract

RAPIDS is a multi-agent system that fully automates the robot data collection and learning pipeline. From a single natural language instruction, 6 AI agents autonomously handle task specification, simulation generation, data collection, quality verification, robot control, and VLA fine-tuning.

Result: 67 GB data across 27 tasks on 3 robot platforms, pipeline time 4–8 weeks → 17 min, cost $10–50 → ~$0.1 per episode.

End-to-end autonomous pipeline: natural language → simulation → data collection → verification → VLA learning.

System Overview

RAPIDS consists of 6 specialized agents orchestrated in an end-to-end pipeline. The system operates across two tracks: Simulation (A-1 → A-2 → A-3 → A-4) for scalable data production in IsaacLab, and Real-World (A-3 → B-1 → B-2) for physical robot execution and VLA model deployment.

NL → Sim → Real → Verified Dataset: A-1 converts natural language to YAML spec → A-2 generates IsaacLab environment → A-3 collects data via Code-as-Policies (sim & real) → A-4 validates quality → B-1 controls heterogeneous robots → B-2 fine-tunes VLA with LoRA — fully autonomous, no human intervention.

Key Differentiators

 End-to-End Autonomy

NL input → 6 agents → trained VLA model. No manual steps. 4–8 weeks → 17 min.

 Closed-Loop Quality

Auto self-refinement, 3-tier validation, AST + VLM dual verification, 10Hz real-time monitoring.

 Physical World Design

6-DOF IK, PhysX 5 physics, auto Domain Randomization, low-level safety limits.

Impact

 Research Productivity

Experiment iteration: weeks → days. 27 tasks, 67 GB data — fully autonomous batch production.

 Lower Barrier

NL-level task definition. 82 YAML templates + 4 robot profiles. Extend via config, not code.

 Open Data

All data in LeRobot v3.0 (OXE compatible) on HuggingFace. Code, models, data — all open.

 Cost Reduction

Per-episode: $10–50 → ~$0.1. Multi-robot, multi-platform deployment ready.
